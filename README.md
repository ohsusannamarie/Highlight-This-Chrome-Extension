# Highlight This

Turn the internet into a searchable layer of memory.

Highlight This is a lightweight Chrome extension that lets you highlight, organize, and revisit important information across the web - without clutter, subscriptions, or bloated note-taking workflows.

Built for researchers, recruiters, sourcers, students, investigators, writers, and people who constantly think:

> "I KNOW I saw that somewhere."

---

## Why This Exists

Most highlighting extensions are either:

- overloaded with unnecessary features
- slow and intrusive
- tied to cloud accounts
- ugly
- abandoned
- or terrible at restoring highlights reliably

Highlight This focuses on speed, simplicity, persistence, and usability.

The goal:
- frictionless highlighting
- reliable restoration
- fast retrieval
- minimal UI clutter
- zero workflow interruption

# Installation

## Load Unpacked (Developer Mode)

1. Clone or download this repository
2. Open Chrome
3. Navigate to: chrome://extensions
4. Enable Developer Mode
5. Click Load unpacked
6. Select the extension folder

Done.
